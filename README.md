# Building
```
git clone https://github.com/smokingplaya/cloud
cd cloud

npm install
npm run build
```

## Installing
* After building, copy the contents of ``cloud/dist/`` into ``GarrysMod/garrysmod/html/``.
* Delete ``main.html``
* Rename ``index.html`` to ``main.html``
